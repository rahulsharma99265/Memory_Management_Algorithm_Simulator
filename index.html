How to Use This Simulator

  1. Set the total memory size and click "Initialize Memory"
  2. Create memory blocks by entering comma-separated block sizes
  3. Enter a Process ID and size, select an algorithm, then click "Allocate Process"
  4. Watch how the algorithm allocates the process to a memory block
  5. Compare different algorithms by allocating the same processes with different methods
